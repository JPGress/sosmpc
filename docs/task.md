# Task List - S.O.S. MPC (MVP)

Esta é a lista numerada de tarefas (`task list`) para a estruturação e implementação do MVP do projeto S.O.S. MPC.

- [x] 1. Inicializar estrutura de diretórios (`core/` e `plugins/`).
- [x] 2. Criar os arquivos de apoio e documentação (`README.md`).
- [x] 3. Implementar módulo de configurações: `core/config.py` (Variáveis, Paths, e Cores ANSI).
- [x] 4. Implementar módulo de log: `core/logger.py` (Registro de atividades).
- [x] 5. Implementar módulo de registro de plugins: `core/registry.py` (Carregamento e catalogação dinâmica de plugins do repositório).
- [x] 6. Implementar módulo de interface: `core/menu.py` (Renderização, cores ANSI baseadas em categorias/grupos de plugin e avisos para instáveis).
- [x] 7. Implementar módulo de roteamento: `core/dispatcher.py` (Direcionamento de escolha de usuário para execução).
- [x] 8. Implementar ponto de entrada: `sosmpc.py` (Inicialização e chamada principal do pipeline MVC).
- [x] 9. Criar estrutura do plugin Guias em `plugins/guias/` com um módulo inicial (ex: `guia_update.py`).
- [x] 10. Criar estrutura do plugin Redes em `plugins/redes/` com um módulo inicial (ex: `info_interfaces.py`).
- [x] 11. Criar estruturas dos plugins de Troubleshooting e Diversos (`plugins/troubleshooting/` e `plugins/diversos/`) com módulos placeholders.
- [x] 12. Validar o funcionamento sistêmico do MVP e realizar a revisão final.
